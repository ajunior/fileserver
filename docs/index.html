fileserver

Servidor de arquivos minimalista baseado em HTTP, desenvolvido em Node.js, motivado pela necessidade de compartilhar arquivos usando um computador que só tem acesso externo a Internet pela porta 80.

As principais funcionalidades incluem navegação entre diretórios, download e upload de arquivos.

Projeto desenvolvido para disciplina de Linguagem Script (JS) do curso superior de Sistema para Internet do Instituto Federal da Paraíba.

O projeto é open source e desenvolvido sob a Licença MIT.

Instalação

A instalação do fileserver é simples e pode ser feita de duas formas:

Git: clonando o repositório

No terminal, acesse o diretório onde deseja clonar o aplicativo e execute o comando:

$ git clone https://github.com/ajunior/fileserver.git
Download: baixando o repositório compactado

Baixe e descompacte no diretório que desejar

Baixar

Requerimentos

O principal requisito para o funcionamento do aplicativo é ter o Node.js instalado e funcionando.

Algumas bibliotecas também são necessárias:

  • Moment.js
  • Sorttable
  • wget-improved
  • dropzone.js

Inicialização

A inicialização do fileserver é simples, basta acessar o diretório do aplicativo e digitar o comando abaixo
indicando o diretório onde os arquivos que serão servidos estão armazenados

$ node app.js rootdir=/path/to/dir

Em seguida, acesso o diretório front-end e abra o arquivo index.html no seu browser favorito
